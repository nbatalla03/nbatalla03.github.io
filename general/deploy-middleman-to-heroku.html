Deploy Middleman To Heroku

Imagine you just typed the last line of code in your latest project and are eager to deploy your creation for the world to see. You settle on Heroku because you hear it's easy, only to find out that it wasn't as straight forward as you were hoping.

Luckily, I already went through the headache of sifting through the multiple setup options available online and laid it out in a simple step by step process.

Prerequisites

-Have your Middleman project in a Github repo.

-Have Bundler installed

-Have a Heroku account

-Have Heroku Toolbelt installed

Heroku Setup

If you haven't already done so, please head on over to Heroku.com and create an account, the signup process is straight forward and FREE!

After you have setup your free Heroku account, install the Heroku Toolbelt, you can download that here: https://toolbelt.heroku.com/. Follow the heroku login instructions under the Getting Started section, but ignore the heroku create instructions for the time being.

Heroku Create

If you haven't already done so, open up your command line terminal and cd into your Middleman project folder.

Once you're in your Middleman project folder type the command heroku create. This will generate a random Heroku URL where your project will be hosted.

Make sure you have committed all your latest changes to your project's Github repo before running the following command in your terminal: git push heroku master. This command will now push the compiled version of your Middleman project onto Heroku's server. This process takes a few seconds to complete, let it run.

Once the process completes you can view your newly deployed Middleman project by typing heroku open or typing the auto generated URL Heroku provided in your browser.
